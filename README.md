# Portfolio - Shamim Alam

A modern, professional portfolio website showcasing web development skills,
projects, and achievements.

## 🎯 Overview

This is an elegant portfolio website built with HTML5, CSS3, and JavaScript. It
features a responsive design, smooth animations, and an interactive user
experience.

**Developer:** Shamim Alam | Front-End Developer

🔗 [Live Demo](https://shamim-01.github.io/Personal_Protfolio1/)

---

## ✨ Features

- **Responsive Navigation** - Mobile-friendly navbar with smooth scrolling
- **Hero Section** - Eye-catching introduction with Three.js background effects
- **About Section** - Personal introduction and background
- **Skills Section** - Technical expertise showcase
- **Achievements Section** - Notable accomplishments and milestones
- **Projects Section** - Portfolio of completed projects
- **Contact Section** - Easy ways to get in touch
- **Resume Button** - Quick access to download resume
- **Smooth Animations** - Professional transitions and effects
- **Mobile Responsive** - Optimized for all screen sizes

---

## 🛠️ Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Advanced layouts, flexbox, grid, and animations
- **JavaScript** - DOM manipulation and interactivity
- **Three.js** - 3D background effects
- **Font Awesome** - Icon library
- **Google Fonts** - Inter and Fira Code typefaces
- **AOS** - Animate On Scroll library
